# (AWS Profile)-Vault

```
USAGE:
    aws-profile <command>... -p <PROFILE>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -p <PROFILE>        The AWS profile to use.

ARGS:
    <command>...    
```
